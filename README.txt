# 이 프로젝트는 CKEditor 5를 사용합니다.
**CKEditor 5**: 39.0.1 (파일로 다운받아 사용)

# 적용법
1. assets/js/ckeditor.js 생성후 
2. 해당주소의 소스코드를 복사후 붙여넣기
    https://cdn.ckeditor.com/ckeditor5/39.0.1/classic/ckeditor.js 
3. 이후 HTML 파일에서 아래와 같이 ckeditor.js 파일을 불러와 사용합니다.
    <script src="상대주소"></script>
    절대주소는 서버에 절대주소와의 충돌을 방지하기위해 상대주소로 지정.
